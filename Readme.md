# 🧮 Calculadora

Uma calculadora web desenvolvida com HTML, CSS e JavaScript, capaz de realizar operações matemáticas básicas de forma simples, rápida e intuitiva.

## ✨ Funcionalidades

* Adição (+)
* Subtração (-)
* Multiplicação (×)
* Divisão (÷)
* Limpar valores
* Interface responsiva
* Design moderno

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Manipulação do DOM
* Eventos JavaScript
* Operações matemáticas
* Estruturação de interfaces responsivas
* Organização de código front-end

## 📂 Estrutura do Projeto

```bash
calculadora/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 💻 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/devKauaHenrique/Calculadora.git
```

2. Acesse a pasta:

```bash
cd Calculadora
```

3. Abra o arquivo `index.html` no navegador.
